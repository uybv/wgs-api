# WGS App API

API for WGS App

## Start for developer

* [NodeJS 12](https://nodejs.org/en/download/package-manager/)

#### Install Serverless framework

```bash
npm install -g serverless
sls login
```

#### Serverless offline

```bash
git clone git@github.com:avadkk/wgs.app-api.git
cd wgs.app-api && npm i
sls dynamodb install
```

#### Run local

```bash
npm run local
```
